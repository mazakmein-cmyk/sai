## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 1920x1080px or larger (16:9 aspect ratio preferred)
- **File Size**: Optimize images to reduce load time (aim for < 500KB per image)
- **Naming**: Use exact filenames as listed above

## Quick Setup

1. Add your temple images to this `public/images/` folder
2. Name them exactly as listed above
3. The carousel will automatically display them
4. If an image is missing, a placeholder will be shown
